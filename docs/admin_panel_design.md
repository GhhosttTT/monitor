# 管理后台设计文档

## 概述

管理后台是一个独立的Web应用程序，为系统管理员提供用户管理、摄像头管理、视频管理和系统监控等功能。它使用现代化的前端框架构建，通过RESTful API与后端服务进行通信。

## 技术栈

- 前端框架: React.js 或 Vue.js
- UI库: Ant Design 或 Element UI
- 状态管理: Redux 或 Vuex
- HTTP客户端: Axios
- 构建工具: Webpack 或 Vite
- 包管理: npm 或 yarn

## 功能模块

### 1. 仪表板

#### 功能描述
展示系统的整体概览信息，包括用户统计、摄像头状态、视频存储情况等。

#### 页面元素
- 系统状态卡片：
  - 总用户数
  - 在线摄像头数
  - 总视频数
  - 存储使用情况
  
- 图表展示：
  - 用户增长趋势图
  - 摄像头状态分布饼图
  - 视频上传量趋势图

- 最近活动列表：
  - 新注册用户
  - 新增摄像头
  - 异常摄像头报告

### 2. 用户管理

#### 功能描述
管理系统中的所有用户，包括普通用户和管理员。

#### 页面元素
- 用户列表表格：
  - 用户ID
  - 用户名
  - 邮箱
  - 角色（用户/管理员）
  - 注册时间
  - 操作（编辑、删除、重置密码）

- 用户搜索和筛选功能：
  - 按用户名搜索
  - 按角色筛选
  - 按注册时间排序

- 用户详情弹窗：
  - 显示用户的详细信息
  - 显示用户拥有的摄像头列表

- 添加/编辑用户表单：
  - 用户名
  - 邮箱
  - 角色选择
  - 密码（仅在添加时显示）

- 重置密码功能：
  - 为用户生成随机密码
  - 发送邮件通知用户

### 3. 摄像头管理

#### 功能描述
管理系统中所有用户的摄像头设备。

#### 页面元素
- 摄像头列表表格：
  - 摄像头ID
  - 名称
  - 序列号
  - 所有者（用户名）
  - 状态（在线/离线/错误）
  - 最后连接时间
  - 操作（编辑、删除）

- 摄像头搜索和筛选功能：
  - 按名称搜索
  - 按状态筛选
  - 按所有者筛选

- 摄像头详情弹窗：
  - 显示摄像头的详细信息
  - 显示摄像头设置（分辨率、存储保留天数等）
  - 显示摄像头状态历史

- 添加/编辑摄像头表单：
  - 名称
  - 序列号
  - 所有者（下拉选择用户）
  - 设置选项（分辨率、存储保留天数等）

### 4. 视频管理

#### 功能描述
管理和查看系统中存储的所有视频记录。

#### 页面元素
- 视频列表表格：
  - 视频ID
  - 文件名
  - 摄像头名称
  - 开始时间
  - 结束时间
  - 时长
  - 大小
  - 分辨率
  - 是否有移动侦测事件
  - 操作（播放、下载、删除）

- 视频搜索和筛选功能：
  - 按摄像头筛选
  - 按时间范围筛选
  - 按分辨率筛选
  - 按移动侦测事件筛选

- 视频播放器：
  - 内嵌视频播放器
  - 支持播放、暂停、快进等基本操作
  - 显示视频元数据

- 批量操作功能：
  - 批量删除视频
  - 批量下载视频

### 5. 系统监控

#### 功能描述
监控服务器运行状态和系统性能指标。

#### 页面元素
- 服务器状态面板：
  - CPU使用率
  - 内存使用情况
  - 磁盘使用情况
  - 网络流量

- 实时连接监控：
  - 当前WebSocket连接数
  - 当前WebRTC连接数
  - 连接趋势图

- 日志查看器：
  - 实时系统日志
  - 错误日志筛选
  - 日志级别筛选（INFO/WARN/ERROR）

- 存储管理：
  - 存储空间使用情况
  - 存储清理功能
  - 存储扩容建议

```
+---------------------------------------------------+
| 顶部导航栏                                       |
| [Logo] [系统名称] [用户菜单]                     |
+-----------+---------------------------------------+
| 侧边栏     | 主内容区域                            |
|           |                                       |
| - 仪表板   |                                       |
| - 用户管理 |                                       |
| - 摄像头管理|                                      |
| - 视频管理 |                                       |

|           |                                       |
+-----------+---------------------------------------+

```

### 主题风格

- 主色调: 深蓝色 (#1890ff) 和白色
- 辅助色: 绿色 (#52c41a) 表示正常状态，红色 (#ff4d4f) 表示异常状态
- 字体: 使用系统默认字体，中文使用苹方或微软雅黑
- 响应式设计: 支持桌面端和移动端浏览

## 权限控制

### 角色定义

1. **超级管理员**
   - 拥有所有功能的完全访问权限
   - 可以管理其他管理员账户

2. **管理员**
   - 可以管理用户、摄像头和视频
   - 可以查看系统监控信息
   - 不能管理其他管理员账户

### 权限验证

- 前端路由权限控制
- 页面元素级权限控制
- API请求权限验证
- 实时权限更新机制

## 安全措施

### 认证安全

- JWT Token认证
- Token过期自动刷新
- 多设备登录管理
- 登录失败次数限制

### 数据安全

- 敏感信息加密存储
- HTTPS通信加密
- 数据备份机制
- 操作日志记录

### 访问安全

- IP白名单限制
- 访问频率限制
- CSRF防护
- XSS防护

## 部署方案

### 环境要求

- Node.js >= 14.x
- npm >= 6.x 或 yarn >= 1.x
- 现代浏览器支持

### 部署步骤

1. 克隆项目代码
2. 安装依赖包
3. 配置环境变量
4. 构建生产版本
5. 部署到Web服务器

### 环境变量

```
REACT_APP_API_URL=http://localhost:5000/api
REACT_APP_WS_URL=ws://localhost:5000
```

## API集成

### 认证相关

- 用户登录: POST /api/auth/login
- 获取用户信息: GET /api/auth/me
- 用户登出: POST /api/auth/logout

### 用户管理

- 获取用户列表: GET /api/users
- 创建用户: POST /api/users
- 更新用户: PUT /api/users/:id
- 删除用户: DELETE /api/users/:id
- 重置密码: POST /api/users/:id/reset-password

### 摄像头管理

- 获取摄像头列表: GET /api/cameras
- 创建摄像头: POST /api/cameras
- 更新摄像头: PUT /api/cameras/:id
- 删除摄像头: DELETE /api/cameras/:id

### 视频管理

- 获取视频列表: GET /api/videos
- 获取视频详情: GET /api/videos/:id
- 删除视频: DELETE /api/videos/:id

### 系统监控

- 获取服务器状态: GET /api/system/status
- 获取实时连接数: GET /api/system/connections
- 获取日志信息: GET /api/system/logs
